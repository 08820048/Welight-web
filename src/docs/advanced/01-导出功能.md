# 导出功能

Welight 支持多种格式的文档导出，满足不同场景的需求。

## 支持的导出格式

### 文档格式
- **PDF**: 高质量的 PDF 文档，保持完美排版
- **Word (.docx)**: Microsoft Word 格式，便于进一步编辑
- **HTML**: 网页格式，可直接在浏览器中查看
- **Markdown (.md)**: 原始 Markdown 格式

### 图片格式
- **PNG**: 高质量的图片格式
- **JPEG**: 压缩的图片格式，文件更小
- **SVG**: 矢量图格式，无损缩放

### 其他格式
- **EPUB**: 电子书格式
- **LaTeX**: 学术论文格式
- **RTF**: 富文本格式

## 导出设置

### PDF 导出选项
```
页面设置:
- 纸张大小: A4, Letter, A3 等
- 页边距: 自定义边距设置
- 页眉页脚: 添加页码、标题等

样式设置:
- 字体选择: 系统字体或自定义字体
- 字体大小: 标题和正文字体大小
- 行间距: 调整行间距和段落间距
- 代码块样式: 语法高亮和背景色
```

### Word 导出选项
- 样式映射: Markdown 元素到 Word 样式的映射
- 图片处理: 图片嵌入或链接方式
- 目录生成: 自动生成文档目录

### HTML 导出选项
- CSS 样式: 内联样式或外部样式表
- 代码高亮: 选择代码高亮主题
- 数学公式: MathJax 或 KaTeX 渲染

## 批量导出

### 多文档导出
1. 选择要导出的多个文档
2. 选择导出格式和设置
3. 一键导出到指定文件夹

### 文件夹导出
- 导出整个文件夹的所有文档
- 保持原有的文件夹结构
- 支持递归导出子文件夹

## 自定义模板

### PDF 模板
```css
/* 自定义 PDF 样式 */
@page {
  margin: 2cm;
  @top-center {
    content: "文档标题";
  }
  @bottom-center {
    content: counter(page);
  }
}

h1 {
  color: #2c3e50;
  border-bottom: 2px solid #3498db;
  padding-bottom: 10px;
}
```

### HTML 模板
```html
<!DOCTYPE html>
<html>
<head>
  <title>{{title}}</title>
  <style>{{styles}}</style>
</head>
<body>
  <div class="container">
    {{content}}
  </div>
</body>
</html>
```

## 导出技巧

### 优化 PDF 输出
- 使用高质量图片
- 选择合适的字体
- 调整页面布局

### 提高导出速度
- 预处理大型图片
- 使用缓存机制
- 分批处理大量文档

### 保持格式一致性
- 使用统一的样式模板
- 检查导出预览
- 测试不同设备的兼容性
