# Markdown 编辑器

Welight 内置强大的 Markdown 编辑器，支持标准 Markdown 语法和扩展功能。

## 基础语法支持

### 标题
```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 文本格式
```markdown
**粗体文本**
*斜体文本*
~~删除线~~
`行内代码`
```

### 列表
```markdown
- 无序列表项 1
- 无序列表项 2

1. 有序列表项 1
2. 有序列表项 2
```

### 链接和图片
```markdown
[链接文本](https://example.com)
![图片描述](image.jpg)
```

## 扩展功能

### 表格支持
```markdown
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 数据1 | 数据2 | 数据3 |
| 数据4 | 数据5 | 数据6 |
```

### 代码块
```markdown
```javascript
function hello() {
    console.log("Hello, World!");
}
```
```

### 数学公式
```markdown
行内公式：$E = mc^2$

块级公式：
$$
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
$$
```

## 编辑器特性

### 实时预览
- 分屏预览模式
- 同步滚动
- 即时渲染

### 智能补全
- 自动补全 Markdown 语法
- 智能缩进
- 括号匹配

### 快捷键
- `Ctrl+B`: 加粗
- `Ctrl+I`: 斜体
- `Ctrl+K`: 插入链接
- `Ctrl+Shift+C`: 插入代码块

## 自定义设置

在设置中，您可以：
- 选择编辑器主题
- 调整字体大小
- 设置行号显示
- 配置自动保存
