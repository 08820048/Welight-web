# 主题系统

Welight 提供丰富的主题选择，让您的写作环境更加个性化。

## 内置主题

### 浅色主题
- **默认浅色**: 经典的白色背景主题
- **GitHub**: 模仿 GitHub 风格的主题
- **Material Light**: Google Material Design 浅色主题

### 深色主题
- **默认深色**: 护眼的深色主题
- **Dracula**: 流行的 Dracula 配色方案
- **One Dark**: Atom 编辑器的经典深色主题
- **Monokai**: Sublime Text 的经典主题

### 特色主题
- **Solarized Light/Dark**: 科学设计的配色方案
- **Nord**: 极地风格的配色主题
- **Gruvbox**: 复古风格的温暖主题

## 主题切换

### 快速切换
1. 使用快捷键 `Ctrl+T` (Windows/Linux) 或 `Cmd+T` (macOS)
2. 从主题列表中选择您喜欢的主题
3. 主题会立即应用到编辑器

### 自动切换
- **跟随系统**: 根据系统的浅色/深色模式自动切换
- **定时切换**: 设置特定时间自动切换主题
- **环境感知**: 根据环境光线自动调整（需要摄像头权限）

## 自定义主题

### 创建自定义主题
1. 进入 设置 → 主题 → 自定义主题
2. 选择基础主题作为模板
3. 调整颜色配置：
   - 背景色
   - 文本色
   - 语法高亮色
   - 界面元素色

### 主题配置文件
```json
{
  "name": "我的主题",
  "type": "dark",
  "colors": {
    "background": "#1e1e1e",
    "foreground": "#d4d4d4",
    "selection": "#264f78",
    "lineHighlight": "#2d2d30",
    "cursor": "#ffffff"
  },
  "syntax": {
    "keyword": "#569cd6",
    "string": "#ce9178",
    "comment": "#6a9955",
    "number": "#b5cea8"
  }
}
```

### 导入导出主题
- **导出**: 将自定义主题保存为 `.wtheme` 文件
- **导入**: 导入其他用户分享的主题文件
- **分享**: 在社区中分享您的主题作品

## 主题市场

### 社区主题
- 浏览社区用户创建的主题
- 下载和安装热门主题
- 为喜欢的主题点赞和评论

### 主题开发
如果您是开发者，可以：
1. 使用主题 API 创建复杂主题
2. 发布主题到主题市场
3. 获得用户反馈和下载统计

## 主题优化建议

### 护眼设置
- 选择对比度适中的主题
- 避免过于鲜艳的颜色
- 在暗光环境下使用深色主题

### 效率提升
- 选择语法高亮清晰的主题
- 确保重要元素有足够的对比度
- 根据写作内容选择合适的主题
