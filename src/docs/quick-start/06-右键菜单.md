## 右键菜单

当您在编辑区点击鼠标右键时，您会看到类似下面这样一个菜单面板，这，就是右键菜单。

![image-20251023154827292](https://images.waer.ltd/notes/202510231548374.png)

如果说前面的斜杆指令是一个常用 markdown 语法快捷输入面板。那么右键菜单则是这个功能的增强扩展版本，通过右键菜单，您可以直接插入所有支持的 markdown 语法格式，包括==表格、代码块、图表、数学公式、脚注、图片、超链接==等。

与此同时，通过右键菜单，您还可以快速打开设置面板，快速查找自己需要的设置，通过左侧菜单并不是唯一的选择。

> [!NOTE]
>
> 同样，随着后续版本的更新迭代，斜杠菜单中将会集成更多的功能，敬请期待。
