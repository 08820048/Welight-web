# 安装指南

本指南将帮助您在不同操作系统上安装 Welight。

## Windows 安装

### 系统要求

- Windows 10 版本 1903 或更高版本
- 至少 4GB RAM
- 500MB 可用磁盘空间

### 安装步骤

1. **下载安装包**
   - 访问 [下载页面](/download)
   - 点击 "Windows 版本" 下载 `.exe` 安装包

2. **运行安装程序**
   - 双击下载的 `Welight-Setup.exe`
   - 如果出现安全提示，点击 "更多信息" → "仍要运行"

3. **完成安装**
   - 选择安装路径（推荐使用默认路径）
   - 勾选 "创建桌面快捷方式"
   - 点击 "安装" 完成

## macOS 安装

### 系统要求

- macOS 10.15 (Catalina) 或更高版本
- Apple Silicon 或 Intel 处理器
- 至少 4GB RAM

### 安装步骤

1. **下载 DMG 文件**
   - 下载 `Welight.dmg` 文件

2. **安装应用**
   - 双击 DMG 文件打开
   - 将 Welight 拖拽到 Applications 文件夹
   - 首次运行时，右键点击选择 "打开"

## Linux 安装

### 支持的发行版

- Ubuntu 18.04+
- Debian 10+
- CentOS 7+
- Fedora 30+

### 使用 AppImage（推荐）

```bash
# 下载 AppImage
wget https://releases.welight.app/latest/Welight.AppImage

# 添加执行权限
chmod +x Welight.AppImage

# 运行应用
./Welight.AppImage
```

### 使用包管理器

#### Ubuntu/Debian

```bash
# 添加 GPG 密钥
wget -qO - https://packages.welight.app/gpg.key | sudo apt-key add -

# 添加软件源
echo "deb https://packages.welight.app/apt stable main" | sudo tee /etc/apt/sources.list.d/welight.list

# 更新并安装
sudo apt update
sudo apt install welight
```

## 验证安装

安装完成后，您可以通过以下方式验证：

1. **启动应用**: 从开始菜单或应用程序文件夹启动 Welight
2. **检查版本**: 在应用中点击 "帮助" → "关于" 查看版本信息
3. **创建测试文档**: 尝试创建一个新的 Markdown 文档

## 常见问题

### Windows 安装问题

**Q: 提示 "Windows 已保护你的电脑"**
A: 点击 "更多信息"，然后选择 "仍要运行"。这是因为应用尚未获得 Microsoft 的数字签名。

**Q: 安装后无法启动**
A: 请确保您的系统满足最低要求，并尝试以管理员身份运行。

### macOS 安装问题

**Q: 提示 "无法打开，因为它来自身份不明的开发者"**
A: 在系统偏好设置 → 安全性与隐私中，点击 "仍要打开"。

### Linux 安装问题

**Q: AppImage 无法运行**
A: 确保已安装 FUSE：

```bash
sudo apt install fuse libfuse2
```

如果您遇到其他问题，请访问我们的 [支持页面](/support) 或 [GitHub Issues](https://github.com/welight/welight/issues)。
