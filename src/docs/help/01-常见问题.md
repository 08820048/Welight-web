### Q: Mac 提示"安装包已损坏"无法打开？

> A: 这是 macOS 的安全机制导致的。请打开终端（Terminal），执行以下命令：
>
> ```bash
> sudo xattr -rd com.apple.quarantine /Applications/Welight.app
> ```
>
> 执行时需要输入管理员密码。执行完成后，即可正常打开 Welight。
