# 常见问题

这里收集了用户最常遇到的问题和解决方案。

## 安装和启动问题

### Q: Windows 上安装后无法启动
**A:** 请尝试以下解决方案：
1. 右键点击 Welight 图标，选择"以管理员身份运行"
2. 检查是否安装了最新的 Visual C++ 运行库
3. 临时关闭杀毒软件，重新安装应用
4. 确保系统满足最低要求（Windows 10 1903+）

### Q: macOS 提示"应用已损坏"
**A:** 这是 macOS 的安全机制，解决方法：
1. 打开终端，运行：`sudo xattr -rd com.apple.quarantine /Applications/Welight.app`
2. 或者在系统偏好设置 → 安全性与隐私中允许运行

### Q: Linux 上 AppImage 无法执行
**A:** 请确保：
1. 文件有执行权限：`chmod +x Welight.AppImage`
2. 安装了 FUSE：`sudo apt install fuse libfuse2`
3. 系统支持 AppImage 格式

## 使用问题

### Q: 如何同步多设备的文档？
**A:** Welight 支持多种同步方式：
1. **云端同步**：登录账户后自动同步到云端
2. **本地网络同步**：在同一网络下的设备间同步
3. **手动导出导入**：使用导出功能在设备间传输

### Q: 文档丢失了怎么办？
**A:** 不要慌张，尝试以下方法：
1. 检查回收站或垃圾箱
2. 查看自动备份文件夹
3. 如果开启了云端同步，检查云端备份
4. 使用文件恢复工具扫描磁盘

### Q: 如何恢复误删的内容？
**A:** Welight 提供多种恢复机制：
1. **撤销操作**：使用 Ctrl+Z 撤销最近的操作
2. **版本历史**：查看文档的历史版本
3. **自动保存**：检查自动保存的临时文件
4. **本地备份**：查看本地备份文件夹

## 性能问题

### Q: 编辑大文件时卡顿
**A:** 优化建议：
1. 关闭实时预览功能
2. 减少同时打开的文档数量
3. 清理应用缓存
4. 增加系统内存

### Q: 启动速度慢
**A:** 可能的解决方案：
1. 关闭开机自启动的其他程序
2. 清理系统垃圾文件
3. 将 Welight 添加到杀毒软件白名单
4. 使用 SSD 硬盘

## 功能问题

### Q: 数学公式不显示
**A:** 检查以下设置：
1. 确保在设置中启用了数学公式支持
2. 检查网络连接（需要加载 MathJax）
3. 尝试刷新预览或重启应用
4. 确认公式语法正确

### Q: 图片无法显示
**A:** 可能的原因：
1. 图片路径不正确（使用相对路径）
2. 图片格式不支持（支持 PNG、JPG、GIF、SVG）
3. 图片文件损坏或过大
4. 权限问题（检查文件访问权限）

### Q: 导出 PDF 失败
**A:** 排查步骤：
1. 检查磁盘空间是否充足
2. 确保有写入目标文件夹的权限
3. 尝试导出到不同的位置
4. 检查文档中是否有特殊字符或格式

## 账户和同步问题

### Q: 忘记了账户密码
**A:** 重置密码：
1. 在登录页面点击"忘记密码"
2. 输入注册邮箱
3. 查收重置密码邮件
4. 按照邮件指引重置密码

### Q: 同步失败
**A:** 检查以下项目：
1. 网络连接是否正常
2. 账户是否已登录
3. 同步服务是否正常（查看状态页面）
4. 本地存储空间是否充足
